Other Games

Minecraft has some other games.

There are many versions of Minecraft.
The most popular ones are Java Edition And Bedrock Edition.

The first Minecraft game to include other gameplay was Minecraft Dungeons
Minecraft Dungeons is a dungeon crawler game with Minecraft Themes,
it also has a topdown view.

There's also a new Minecraft Game.
Its called Minecraft Legends.
Minecraft Legends is a real-time action-strategy game.
The game includes singleplayer and multiplayer.
